# Velm Documentation

Welcome to the Velm documentation.

These docs explain the **core concepts, architecture, and usage patterns**
behind Velm’s logical models and services.

If you are new to Velm, start with **Getting Started**.
If you are evaluating Velm, begin with **Concepts → Pipelines**.
